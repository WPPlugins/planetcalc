Planetcalc Plugin allows to embed planetcalc.com calculators into your wordpress site.

== Description ==

[planetcalc.com](http://planetcalc.com/ "PLANETCALC") contains hundreds of different online calculators in 3 languages (English, Spanish, Russian).
The **online calculator** is a simple piece of software written in javascript intended to automate routine calculations.
With Planetcalc Plugin for Wordpress you may place any of planetcalc calculators on your site pages. 
All the calculators are distributed under Creative Commons Attribution/Share-Alike License 3.0 (Unported) license.

== Installation ==

1. Unpack all the plugin files to the `/wp-content/plugins/planetcalc/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use insert PLANETCALC calculator button in your WP post editor to select a calculator for embedding
